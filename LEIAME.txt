Para iniciar o programa é necessário primeiro abrir a pasta no terminal. Após abrir a pasta é necessário compilar o código, para efetuar a compilação é so digitar o seguinte comando:

javac *.java


Após executar esse comando o programa ja está pronto para ser iniciado pelos comandos a seguir:

java Main 0

ou

java Main 1


O comando contendo o "0" irá executar todas as listas utilizando a implementação da pilha ingênua, mas caso o comando contenha "1" irá executar utilizando a implementação da pilha arranjo correta.

Assim o programa irá processar todas as listas contidas na pasta entradas e gerar um arquivo de saída para cada arquivo de entrada, alem de gerar um arquivo chamado compilado, que possui o tempo que o programa demorou para executar cada implementação e a soma final do tempo.

Os arquivos gerados ficarão guardados na pasta saidas.
